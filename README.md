# megalodonte-libs

## clone all the submodules

```bash
git clone --recurse-submodules https://github.com/eliezer-dev-software-enginner/megalodonte-libs.git
```

## Enter on `megalodonte-libs`

```bash
cd megalodonte-libs
```

## run on linux

```bash
./install-all.sh
```

## run on windows

```bash
./install-all.ps1
```
